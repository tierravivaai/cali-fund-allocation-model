# IUSAF Outputs (v3)

Static allocation outputs generated in March 2026 during v3 development. These CSVs were the original figure and annex data before the Streamlit sensitivity app made on-the-fly generation possible.

## Files

| File | Description |
|------|-------------|
| `fig1_gini_spearman_data.csv` | Gini and Spearman data for Figure 1 |
| `fig2_per_party_allocations_data.csv` | Per-Party allocation data for Figure 2 |
| `fig3_band_allocations_data.csv` | Band allocation data for Figure 3 |
| `fig4_sids_tsac_sosac_interaction_data.csv` | SIDS/TSAC/SOSAC interaction data for Figure 4 |
| `annex_all_parties_all_fund_sizes.csv` | All-party annex across fund sizes |
| `annex_summary_by_fund_size.csv` | Summary table by fund size |

Also contains `IUSAF_Cali_Fund_allocation_options.docx` (not tracked — gitignored).

Superseded by: the Streamlit sensitivity app (`src/sensitivity.py`) which generates equivalent data on-the-fly, and the v4 sensitivity reports.
