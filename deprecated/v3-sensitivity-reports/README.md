# v3 Sensitivity Reports

Sensitivity outputs generated in March 2026 for the v3 model (pre-Option D threshold revision). These use the earlier Spearman ρ ≥ 0.85 threshold and the `gini_optimal` naming (later renamed to `gini_minimum`).

## Files

| File | Description |
|------|-------------|
| `scenario_metrics.csv` | Per-scenario Gini, Spearman, overlay strength metrics |
| `balance_point_summary.md` | Balance-point narrative summary |
| `balance_points.csv` | Identified balance-point values |
| `country_results.csv` | Per-country allocation results |
| `country_deltas.csv` | Country-level changes vs pure IUSAF |
| `group_summary.csv` | Group-level (LDC, SIDS, income) totals |
| `integrity_checks.csv` | Invariant checks per scenario |
| `local_stability.md` | Local stability narrative |
| `local_stability_checks.csv` | Per-parameter stability diagnostics |
| `one_way_sweep.csv` | One-way sweep results |
| `sosac_fine_sweep.csv` | Fine-grained SOSAC sweep |
| `tsac_fine_sweep.csv` | Fine-grained TSAC sweep |
| `scenario_brief.md` | Scenario brief narrative |
| `sweep_summary.md` | Sweep summary narrative |
| `technical_annex.md` | Technical annex narrative |
| `comparative_report.md` | Comparative report narrative |

Superseded by: `sensitivity-reports/v4-sensitivity-reports/` which uses the band-order preservation constraint and updated naming.
