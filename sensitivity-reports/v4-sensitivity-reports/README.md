# v4 Sensitivity Reports

Robustness diagnostics and parameter-sweep outputs for the IUSAF allocation model. Generated by the Streamlit sensitivity app (`src/sensitivity.py`) and the calibration harness (`scripts/calibrate_banded_tsac.py`).

## Reports

| File | Description |
|------|-------------|
| `Cali_Fund_v4_sensitivity_reports.docx` | Compiled DOCX of all reports |
| `v4_sensitivity_reports_compiled.md` | Full compiled markdown |
| `executive_summary.md` | High-level findings |
| `technical_annex.md` | Detailed methodology and results |
| `two_way_grid_analysis.md` | TSAC×SOSAC grid analysis |
| `scale_invariance.md` | Scale invariance verification |
| `integrity_checks_report.md` | Per-scenario invariant checks |

## Two-Way TSAC×SOSAC Grids

| File | Scenarios | Resolution |
|------|-----------|------------|
| `tsac_sosac_grid_coarse.csv` | 176 | TSAC 0–15% (1pp), SOSAC 0–10% (1pp) |
| `tsac_sosac_grid_fine.csv` | 441 | TSAC 0–10% (0.5pp), SOSAC 0–10% (0.5pp) |

Columns per row: `scenario_id`, `tsac_beta`, `sosac_gamma`, `gini_coefficient`, `spearman_vs_pure_iusaf`, `top20_turnover_vs_pure_iusaf`, `overlay_strength_label`, and more.

## Heatmaps

- `tsac_sosac_heatmap_gini_{coarse,fine}.png` — Gini coefficient heatmaps
- `tsac_sosac_heatmap_spearman_{coarse,fine}.png` — Spearman ρ vs pure IUSAF heatmaps

## Integrity Checks

`integrity_checks.csv` — one row per scenario, `all_checks_pass=PASS` for all standard scenarios.

## Calibration Subdirectory

`calibration/` contains outputs from the banded TSAC calibration harness, comparing 7 band-weight configurations (linear baseline, linear progression, geometric base 1.5/2/3, capped top, flat) against acceptance criteria. Each configuration has a grid CSV (176 scenarios) and integrity CSV. Summary in `calibration/calibration_summary.md`.

## Regeneration

These reports can be regenerated interactively via the Streamlit app:

```bash
streamlit run src/sensitivity.py
```

Or via the calibration harness:

```bash
python scripts/calibrate_banded_tsac.py
```
