# Model Tables

Generated output tables (CSV) used in the working paper and validated against the live calculator.

## Structure

```
model-tables/
├── iusaf-unregion-*.csv              # Allocation by UN Region
├── iusaf-unsubregion-*.csv           # Allocation by UN Sub-Region
├── iusaf-unintermediate-region-*.csv # Allocation by Intermediate Region
├── iusaf-ldc-panel-*.csv            # LDC allocation panel
├── iusaf-sids-panel-*.csv           # SIDS allocation panel
├── iusaf-sids-countries-*.csv       # SIDS band distribution
├── iusaf-ranked-country-*.csv       # Top 31 / Bottom 22 / Middle 89 parties
├── iusaf-alphabetical-country-*.csv # Alphabetical country listing
├── iusaf-band-order-preservation.csv # Band-order preservation across TSAC levels
├── iusaf-breakpoint-summary.csv     # TSAC break-point summary
├── iusaf-strict-ranked-country.csv  # Strict balance-point ranked countries
├── iusaf-gini_minimum-ranked-country.csv # Gini-minimum ranked countries
├── iusaf-band_order_boundary-ranked-country.csv # Band-order boundary ranked countries
├── iplc-option1-equality.md         # IPLC Option 1 (equality) summary
├── iplc-option2-banded.md           # IPLC Option 2 (banded IUSAF) summary
└── table-validation/                # Validation CSVs + report
    ├── iusaf-*-valid.csv            # One valid.csv per source CSV
    └── validation-report.json       # Aggregate pass/fail report
```

## Generation

Tables are generated by the scripts in `scripts/`:

| Script | Output |
|--------|--------|
| `scripts/validate_all_tables.py` | Region, sub-region, intermediate region, LDC/SIDS panels, ranked country tables |
| `scripts/generate_optiond_tables.py` | Band-order preservation, breakpoint summary |
| `scripts/generate_balance_point_rankings.py` | Balance-point ranked country tables |
| `scripts/rank_panels_scenarios.py` | Scenario comparison panels |
| `scripts/rank_change_scenarios.py` | Rank-change scenarios |

## Validation

Each CSV in this directory has a corresponding `-valid.csv` in `table-validation/`, generated by re-running the calculator and comparing outputs. Run validation with:

```bash
python scripts/validate_all_tables.py
```

See the [Table Sources](../README.md#table-sources) section in the main README for the mapping between paper tables and these CSVs.
