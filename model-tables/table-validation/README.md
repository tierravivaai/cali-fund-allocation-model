# Table Validation

Validation CSVs and report generated by `scripts/validate_all_tables.py`. Each source CSV in `model-tables/` has a corresponding `-valid.csv` here, produced by re-running the live calculator and comparing outputs.

## Files

| Pattern | Description |
|---------|-------------|
| `iusaf-*-valid.csv` | One validation CSV per source table |
| `validation-report.json` | Aggregate pass/fail report |

## Regeneration

```bash
python scripts/validate_all_tables.py
```

See `docs/table-validation-2026-05-04.md` for the latest validation report narrative.
