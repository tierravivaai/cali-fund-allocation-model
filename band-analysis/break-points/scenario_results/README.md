# Scenario Results

Per-scenario allocation CSVs for the break-point analysis. Each file contains ranked country allocations for one named scenario.

| File | Scenario | Description |
|------|----------|-------------|
| `pure_iusaf.csv` | IUSAF (Pure) | TSAC=0%, SOSAC=0% — baseline |
| `strict.csv` | Strict | TSAC=1.5%, SOSAC=3% |
| `gini_minimum.csv` | Gini-minimum | TSAC=2.5%, SOSAC=3% |
| `bounded.csv` | Bounded | TSAC=3.5%, SOSAC=3% |
| `band_order_overturn.csv` | Band-order overturn | TSAC=3.0%, SOSAC=3% — Band 6 catches Band 5 |
| `sosac_overturn.csv` | SOSAC overturn | SOSAC at overturn level |
| `tsac_overturn.csv` | TSAC overturn | TSAC at overturn level |

Generated by `analysis.py`.
