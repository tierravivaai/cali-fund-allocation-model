# Band Mobility History

Analysis of historical band transitions between UN Scale assessment periods and the risk of band crossover under the IUSAF allocation model.

## Files

| File | Description |
|------|-------------|
| `historical_band_mobility.csv` | Band transitions per Party across UN Scale assessment periods |
| `historical_band_mobility.md` | Analysis of historical mobility patterns |
| `band_crossover_risk.csv` | Per-Party risk of crossing band boundaries under future scale revisions |
| `band_crossover_risk.md` | Analysis of crossover risk and implications for allocation stability |
| `near_term_band_movement_note.md` | Note on near-term band movement expectations |
